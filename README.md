Paquete de seguridad para PHP.

## Instalación

Usa Composer para instalar el paquete:

```bash
composer require xvii/satelles-securitas
```

## Requisitos

- PHP >= 8.1
- Extensiones: simplexml, libxml, dom

## Autoload

El paquete utiliza PSR-4:

```
Tabula17\Satelles\Securitas\
```
